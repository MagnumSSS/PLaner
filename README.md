Своего рода планировщик задач, пишем категорию и задачу к ней, каждая задача может стать категорией и наоборот. Реализована на связных списках, имееет при себе Сохранение в файл, Загрузку из Файла, CLI UI
Команды:

добавить X <- добавляет X, как самую первую категорию
добавить Y в X <- добавляет задачу/категорию Y в категорию X
удалить X <- удаляет категорию X и все остальные ветки/задачи вместе с ней
удалить X в Y <- удаляет подкатегорию/задачу Y и все ее остальные ветки/задачи вместе с ней в категории X
показать план - показывает конечную картину, например
выход - выйти из планировщика задач

Загрузить с файла? (y/n): n
Сохранять расчёты в файл? (y/n): n
